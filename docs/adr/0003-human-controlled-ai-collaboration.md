# ADR 0003: 사용자 통제형 AI 협업 개발 방식 채택

## 상태

승인됨

## 맥락

이 프로젝트의 목적은 앱 완성뿐 아니라 전체 개발 프로세스 이해, 코드 리뷰 능력, 학생 수업 확장 가능성 확보에 있다.

## 결정

AI는 자동 구현자가 아니라 설계 보조자, 코드 초안 작성자, 오류 분석자, 리뷰어로 사용한다. 최종 통제권은 사용자에게 있다.

## 대안

- AI에게 전체 앱 자동 생성
- AI를 거의 사용하지 않고 수동 구현
- 사용자 통제형 AI 협업

## 결과

개발 속도는 느려질 수 있으나 유지보수 가능성과 교육적 가치가 높아진다.
